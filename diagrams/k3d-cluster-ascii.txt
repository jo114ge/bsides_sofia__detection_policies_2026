  Container roles
  ---------------
  server-0    Control plane. Runs the Kubernetes API server,
              scheduler, and etcd (state store).

  agent-0     Worker node. This is where your pods actually run:
              demo-app, Kyverno, Argo CD, Prometheus, Grafana.

  serverlb    Load balancer (traefik-proxy). Exposes ports
              80, 443, and 8443 from the cluster to localhost.

  tools       Short-lived helper container used by k3d during
              cluster creation. Exits automatically once done.


  Port bindings (host -> cluster)
  --------------------------------
  localhost:8443  -->  Kubernetes API server  (kubectl)
  localhost:80    -->  cluster HTTP ingress
  localhost:443   -->  cluster HTTPS ingress

  Port-forwards added manually during workshop:
  localhost:8081  -->  argocd-server svc  (Argo CD UI)
  localhost:3000  -->  monitoring-grafana svc  (Grafana UI)
